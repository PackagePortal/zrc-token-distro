
## requirements ##
- node 12.12

## instructions  ##
- `git clone https://github.com/PackagePortal/zil-token-distro.git`
- `cd zil-token-distro`
- `npm install`

## run example-cron ##
- `node example-cron`

## run merkle & claim example ##
- `node example`

## run tests ##
- `./node_modules/mocha/bin/mocha tests -s 0` (`tests/test.js`)
